# uGO Playground

## Project setup

Install followings:

- go
- node
- npm

```
go get github.com/ozanh/ugo
```

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Test
